# FE-Library - ห้องสมุดความรู้ Frontend พื้นฐาน

โปรเจกต์นี้รวบรวมองค์ความรู้พื้นฐานของ Frontend Development สำหรับผู้เริ่มต้น

## เนื้อหาที่รวมอยู่

### 1. HTML พื้นฐาน
- โครงสร้าง HTML ที่ถูกต้อง
- HTML Semantic Elements
- Forms และ Input Elements
- Lists, Tables, และ Media Elements

### 2. CSS พื้นฐาน
- CSS Selectors
- Box Model
- Flexbox Layout
- Responsive Design
- Colors และ Typography

### 3. Website Layouts
- Layout Patterns 9 แบบ
- Visual Diagram แต่ละ Layout
- โค้ดตัวอย่างครบทุก Pattern
- เปรียบเทียบและ Tips การเลือก Layout

### 4. Next.js Client Components ⭐ ใหม่!
- Client Component คืออะไร
- Server vs Client Component
- ตัวอย่างการใช้งานจริง (Counter, Form, Timer, Todo List)
- Interactive Demo ที่ทดลองได้เลย
- Best Practices และ Common Patterns

## วิธีใช้งาน

1. เปิดไฟล์ `.html` ด้วย Web Browser โดยตรง
2. ศึกษาโค้ดและคำอธิบายใน Comments
3. ลองแก้ไขโค้ดเพื่อทดลองการทำงาน

## การเริ่มต้น

เริ่มจาก `01-html-basics/index.html` เพื่อเรียนรู้โครงสร้าง HTML พื้นฐาน
